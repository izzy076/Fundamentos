¿ Qué es el HTML?

Lenguaje de Marcado que estructura el contenido de una página web. Basado en etiquetas que pueden ser semánticas o no semánticas

¡No es un lenguaje de programación!

Semántica en HTML

Etiquetas básicas del HTML

Etiqueta Descripción
h1, h2, h3, h4, h5, h6 Títulos y subtítulos en HTML
p Párrafos
img Para mostrar las imágenes (src, alt)
ul, ol Etiquetas para definir listas desordenadas y ordenadas
div Divisores de contenido - dar estilos
a Es redireccionar a un link fuera o dentro de nuestra página

Formularios de HTML

Son una herramienta fundamental para recopilar datos de un usuario, a través de campos como:

  • Textos
  • casillas de texto - áreas de texto
  • listas desplegables
  • botones
